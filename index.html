Mazebuilder Program

Davis, CA

Utilized C programming to create a mazebuilder program, which lets a user turn a grid into a maze by breaking the user desired grid walls. The program starts by asking the user for the number of rows and columns desired, and then generates a grid accordingly. Next, the program asks the user for two cell numbers (each cell is designated by a row and column number) which holds the wall that should be broken. If the two cells are directly next to each other (and not diagonal), the grid should be regenerated with the desired wall deleted. If the two cells are not adjacent, there will be a notification and a prompt to try again.

The codes were uploaded to GitHub.
